# Glossary — Canonical Terms

Canonical terms used across all plugins. Forbidden synonyms must not appear in operational files.

| Concept | Canonical Term | Forbidden Synonyms |
|---------|---------------|-------------------|
| Web research agent | general-purpose researcher (with WebSearch) | web researcher, web-research agent |
| Code examples for coders | gold standard | gold standard file, reference file, canonical example |
| Review for SIMPLE tasks | unified reviewer | combined reviewer, single reviewer |
| Review for MEDIUM/COMPLEX | specialized reviewers (security + logic + quality) | full review pipeline, 3-reviewer pipeline |
| Deeper review on COMPLEX | deeper scrutiny | deep-analysis agents, deep analysis |
| Agent team orchestrator | Lead | team lead (lowercase), moderator (in build context) |
| Operational monitor | Supervisor | monitor agent, watcher |
| Implementation agent | Coder | developer, implementer |
| Plan validator | Tech Lead | architect |

## Capitalization Rules
- "gold standard" — lowercase (not "Gold Standard") except at sentence start
- "Lead", "Supervisor", "Tech Lead", "Coder" — capitalized when referring to the role
- "SIMPLE", "MEDIUM", "COMPLEX" — all caps for complexity levels
